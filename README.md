# Paper Doll House

A 2D house editor in a paper-doll style. Add overlapping house parts, switch backgrounds/weather, and arrange everything freely on a blank canvas.

## Features

- Fuzzy, cutout-style pieces with window-cling vibes
- Background color and weather picker
- Foundation, base, bricks, trusses, windows, roof, doors, and more
- Pieces can overlap and be dragged around without snapping together

## Run

Open `index.html` in a browser or serve the folder as static files.
